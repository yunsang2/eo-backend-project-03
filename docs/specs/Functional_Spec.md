# AI 채팅 서비스 기능 명세서

## 1. 회원 관리 및 인증 기능
- 회원가입 및 이메일 인증
  - 사용자는 이메일 인증을 완료해야 가입 절차를 진행할 수 있다.
  - 가입 시 비밀번호는 반드시 암호화하여 저장하며, 초기 BASIC 플랜과 기본 토큰을 부여한다.
- 로그인 및 세션 유지
  - JWT(JSON Web Token)를 사용하며, 보안을 위해 Access Token과 Refresh Token을 HttpOnly 쿠키에 저장한다.
  - 로그아웃 시 서버는 해당 쿠키의 만료 시간을 0으로 설정하여 즉시 삭제한다.
- 비밀번호 재설정
  - 비밀번호 분실 시 이메일로 임시 인증 토큰을 발행한다.
  - 발행된 토큰은 일정 시간이 지나면 만료되어야 하며, 변경 완료 시 토큰을 폐기한다.

## 2. 플랜 및 구독 관리 기능
- 등급별 기능 제한
  - 시스템은 사용자의 플랜 등급에 따라 접근 가능한 API 엔드포인트를 제어한다.
  - BASIC: 일반 채팅만 허용.
  - PRO: 일반 채팅 및 웹 페이지 요약 기능 허용.
  - PREMIUM: 모든 기능(유튜브 요약 포함) 허용.
- 플랜 만료 및 강등
  - 유료 플랜(PRO, PREMIUM)의 기본 유지 기간은 결제 후 30일이다.
  - 시스템은 만료일을 체크하여 기간이 지난 사용자를 자동으로 BASIC 플랜으로 강등시킨다.
- 토큰 자동 초기화
  - 매월 또는 특정 주기에 맞춰 사용자의 잔여 토큰을 현재 플랜의 한도량만큼 재충전한다.

## 3. AI 채팅 및 토큰 소모 기능
- 채팅 세션 관리
  - 사용자는 새로운 대화를 시작하거나 기존 세션 ID를 이어받아 대화를 계속할 수 있다.
  - 세션 삭제 시 해당 대화와 연관된 모든 메시지 데이터는 연쇄 삭제된다.
- 실시간 토큰 차감
  - AI 모델의 응답이 완료되면 사용된 토큰 수만큼 사용자의 잔여 토큰에서 즉시 차감한다.
  - 잔여 토큰이 부족한 경우 AI 호출 요청을 거부하고 오류 메시지를 반환한다.

## 4. 결제 및 검증 기능
- 외부 결제 연동
  - 포트원(Portone) SDK를 사용하며, 결제에 필요한 설정값은 서버에서 동적으로 내려받는다.
- 결제 위변조 방지
  - 결제 완료 후 서버는 포트원 API를 호출하여 실제 승인된 금액과 요청된 플랜의 금액이 일치하는지 검증한다.
  - 검증이 완료된 건에 한해서만 유저의 플랜 승급 및 토큰 충전을 처리한다.

## 5. 관리자 제어 기능
- 사용자 제어
  - 관리자는 불량 사용자의 계정을 잠금(LOCKED) 처리하여 서비스 이용을 중단시킬 수 있다.
- 정책 관리
  - 관리자는 플랜별 토큰 한도나 제공 모델 목록을 관리자 대시보드에서 실시간으로 수정할 수 있다.
- 시스템 모니터링
  - 플랜별 가입 현황 및 모델별 토큰 소모량을 통계 데이터로 확인하여 운영 지표로 활용한다.
