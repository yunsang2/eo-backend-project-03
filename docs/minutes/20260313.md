# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 13일
- 참여자: 현길용, 김희성

## 오전 회의 내용

- 관리자 통계 API(플랜별 사용자, 모델별 사용량) 로직 최종 점검 및 에러 수정
- 앨런 AI(Alan AI) 연동을 위한 API 명세서 검토 및 DTO 구조 확정

## 작업 목표

- 앨런 AI API 연동: RestTemplate을 활용한 실시간 AI 질문/답변 통신 모듈 구현
- 채팅 도메인 고도화: ChatSession 및 MessageEntity를 활용한 대화 이력 저장 및 통계 데이터 연동

## 작업 과정

- AdminController 내 중복 매핑(Ambiguous mapping) 에러 디버깅 및 경로 정리 완료
- UserRepository 및 MessageRepository에 JPA Projection을 도입하여 플랜별/모델별 집계 쿼리 구현
- AdminControllerTest에 통계 응답 검증 케이스를 추가하여 관리자 기능의 신뢰성 확보

## 작업 회고

- 컨트롤러 설계 시 동일한 경로(`stats/usage`)에 대한 중복 정의가 서버 기동을 방해하는 이슈를 해결하며 엔드포인트 관리의 중요성을 확인
- 인터페이스 기반 Projection을 통해 복잡한 통계 데이터를 별도의 수동 변환 없이 DTO로 깔끔하게 처리하는 구조 안착

## 내일 목표

- 앨런 AI API 실제 호출 테스트 및 답변 내 토큰 정보 파싱 로직 구현
- AI 답변 수신 직후 유저의 잔여 토큰을 즉시 차감하는 실시간 반영 기능 개발
