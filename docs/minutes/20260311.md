# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 11일
- 참여자: 윤상이 (팀장), 현길용, 김희성

## 오전 회의 내용

- 작성하던 부분 마무리
- 프론트 메인화면 하던 작업 이어서 진행

## 작업 목표

- 앨런 AI API 연동: RestTemplate 또는 WebClient를 활용하여 실제 AI 질문/답변 기능 구현
- 채팅 도메인 개발: ChatSession 및 ChatMessage 엔티티 설계와 토큰 실시간 차감 로직 연동
- 메인페이지 마무리 작업

## 작업 과정

- UserService 및 AdminService 핵심 로직(수정, 탈퇴, 상태 변경)에 대한 Mockito 단위 테스트 작성 및 통과
- UserEntity와 PlanEntity 내부에 상태 제어 및 설정 업데이트를 위한 비즈니스 메서드 구현
- JwtAuthenticationFilter에서 UserMapper 호출 시 UUID(ID) 인자 불일치 에러 해결 및 인증 객체 고도화


## 작업 회고

- CustomUserDetails에 고유 ID를 포함시켜 컨트롤러에서 유저 식별자를 즉시 꺼내 쓰는 보안 체계 구축

## 내일 목표

- 앨런 AI API 통신을 위한 AlanAiClient 구현 및 실시간 토큰 차감 기능 테스트
- 채팅 세션 생성 및 과거 대화 내역 조회 API 개발 완료
